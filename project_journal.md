📌 Project Plan & Development Roadmap

Project Name

Bank Loan Risk Analytics
An End-to-End Banking Analytics Project using Python, SQL, Power BI, Machine Learning, and Streamlit.

⸻

Project Objective

The objective of this project is to build a complete Banking Loan Risk Analytics platform capable of analyzing historical loan data, identifying business insights, monitoring loan portfolio performance, predicting loan default risk, and presenting results through an interactive Power BI dashboard and a Streamlit web application.

This project is designed to demonstrate the practical skills expected from a modern Data Analyst in 2026 by combining business analytics, SQL, visualization, and lightweight machine learning into one end-to-end solution.

### Entry 011

Date: Phase 8 - Model Optimization Completed

Completed

- Evaluated multiple classification thresholds.
- Compared the original Logistic Regression, optimized Logistic Regression, and Random Forest models.

Final Model Performance

- Precision: 23.43%

- F1 Score: 32.83%

Engineering Decision

The optimized Logistic Regression model was selected because it provides the best balance between predictive performance and interpretability. While Random Forest achieved higher overall accuracy, it failed to identify default borrowers effectively. The final model is therefore better aligned with the business objective of credit risk prediction.
